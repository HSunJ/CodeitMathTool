# API
## 계산기(calculator.js)
### add
* 파라미터
    - a(Number) : 덧셈하고자 하는 수
    - b(Number): 덧셈하고자 하는 수
* 응답
	- data(Number): 두 숫자를 덧셈한 결과값
### substract
* 파라미터
    - a(Number) : 뺄셈하고자 하는 수
    - b(Number) : 뺄셈하고자 하는 수
* 응답
	- data(Number): 두 숫자를 뺄셈한 결과값
### multiply
* 파라미터
  - a(Number): 곱셈하고자 하는 수
  - b(Number): 곱셈하고자 하는 수
* 응답
	- data(Number): 두 숫자를 뺄셈한 결과값
### divide_codeit
* 파라미터
  - a(Number): 나눗셈하고자 하는 수
  - b(Number): 나눗셈하고자 하는 수
* 응답
	- data(Number): 두 숫자를 나눗셈한 결과값
